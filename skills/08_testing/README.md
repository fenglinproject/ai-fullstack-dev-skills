# 08_testing Skills

本目录包含以下细粒度 Skill：

- [测试计划](./00_test_plan/SKILL.md)：为项目生成测试范围、策略、环境、人员、时间和通过标准
- [功能测试用例](./01_functional_test_cases/SKILL.md)：按 PRD 逐条生成正常流程功能测试用例
- [边界与异常用例](./02_boundary_exception_cases/SKILL.md)：生成空值、超长、重复、非法状态、第三方失败等测试
- [权限测试用例](./03_permission_test_cases/SKILL.md)：生成角色、菜单、按钮、数据范围、越权访问测试
- [接口测试用例](./04_api_test_cases/SKILL.md)：为接口生成参数、返回、错误码、权限、分页、排序测试
- [兼容性测试](./05_compatibility_test_cases/SKILL.md)：覆盖浏览器、安卓/iOS、屏幕尺寸、小程序版本等
- [性能测试计划](./06_performance_test_plan/SKILL.md)：设计页面加载、接口响应、并发、慢查询、缓存命中测试
- [Bug 报告模板](./07_bug_report_template/SKILL.md)：把发现的问题整理为可复现、可修复、可回归的报告
- [回归测试套件](./08_regression_test_suite/SKILL.md)：按核心链路和修复点生成回归测试清单
- [上线验收测试](./09_release_acceptance_test/SKILL.md)：上线前最终检查功能、数据、权限、安全、部署
