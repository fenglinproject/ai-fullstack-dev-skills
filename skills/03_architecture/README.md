# 03_architecture Skills

本目录包含以下细粒度 Skill：

- [技术栈选型](./00_tech_stack_selection/SKILL.md)：为单人全栈项目选择低成本、高效率、易维护技术栈
- [整体架构方案](./01_architecture_blueprint/SKILL.md)：设计前后端、数据库、缓存、存储、第三方服务的整体架构
- [项目目录结构规范](./02_repo_structure_standard/SKILL.md)：规划前端、后端、公共模块、配置、文档目录
- [环境规划](./03_environment_strategy/SKILL.md)：规划本地、测试、生产环境与变量隔离
- [认证架构](./04_auth_architecture/SKILL.md)：设计登录、注册、Token、刷新、退出、多端登录
- [权限模型](./05_permission_model/SKILL.md)：设计 RBAC、数据权限、菜单权限、按钮权限
- [接口规范](./06_api_standard/SKILL.md)：定义 REST/RPC 风格、状态码、返回体、分页、筛选、错误码
- [异常处理规范](./07_error_handling_standard/SKILL.md)：设计前后端统一错误处理与用户提示
- [日志与监控方案](./08_logging_monitoring_plan/SKILL.md)：规划访问日志、业务日志、错误日志、审计日志、监控指标
- [缓存策略](./09_cache_strategy/SKILL.md)：设计 Redis/本地缓存、失效、穿透、击穿、雪崩防护
- [第三方服务方案](./10_third_party_service_plan/SKILL.md)：规划登录、支付、短信、邮件、地图、OCR、OSS 等接入
- [安全架构方案](./11_security_architecture/SKILL.md)：在设计阶段规划防注入、防 XSS、防越权、鉴权、限流等
