# 06_frontend Skills

- [表单校验与提交](./06_form_validation_submit/SKILL.md)：实现表单校验、错误提示、提交、防重复点击
- [列表搜索筛选分页](./07_list_search_pagination/SKILL.md)：实现列表展示、搜索、筛选、排序、分页、刷新
- [空/加载/错误状态处理](./08_error_empty_loading_states/SKILL.md)：统一处理接口失败、空数据、无权限、超时、重试
- [多端适配](./09_mobile_pc_adaptation/SKILL.md)：处理移动端、PC、不同屏幕、浏览器兼容
- [前端自测](./10_frontend_self_test/SKILL.md)：生成页面渲染、交互、权限、异常、兼容自测清单
