# 12_code_quality Skills

本目录包含以下细粒度 Skill：

- [代码审查](./00_code_review/SKILL.md)：从需求、逻辑、安全、性能、可维护性角度审查代码
- [重构计划](./01_refactoring_plan/SKILL.md)：识别重复、混乱、耦合、难维护代码并制定低风险重构步骤
- [Git 分支提交规范](./02_git_branch_commit/SKILL.md)：规划 dev/test/prod 分支、提交信息、合并与冲突处理
- [依赖升级检查](./03_dependency_upgrade/SKILL.md)：检查依赖版本、漏洞、兼容性、升级步骤
- [性能优化](./04_performance_optimization/SKILL.md)：分析页面慢、接口慢、SQL 慢、缓存低效等问题
- [可观测性与日志规范](./05_observability_logs/SKILL.md)：规范 requestId、用户、接口、耗时、错误、审计日志
