# 11_iteration Skills

本目录包含以下细粒度 Skill：

- [用户反馈归类](./00_feedback_triage/SKILL.md)：把用户反馈分为 bug、需求、体验、数据、误解
- [版本规划](./01_version_planning/SKILL.md)：规划小版本迭代，控制范围，优先修 bug 再做高价值功能
- [线上紧急修复](./02_hotfix_workflow/SKILL.md)：定位线上问题、热修复、回归、发布、记录台账
- [需求池维护](./03_backlog_grooming/SKILL.md)：维护待办需求优先级、价值、成本、风险
- [版本发布说明](./04_release_notes/SKILL.md)：生成用户可读和内部可追踪的发布说明
- [交付文档沉淀](./05_handover_docs/SKILL.md)：整理 PRD、设计、接口、数据库、部署、用户手册
