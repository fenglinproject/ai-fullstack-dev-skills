# 00_core Skills

本目录包含以下细粒度 Skill：

- [项目总控官](./00_project_orchestrator/SKILL.md)：把一个人全栈开发拆成阶段、任务、验收点，并控制节奏与范围
- [项目上下文收集](./01_project_context_collector/SKILL.md)：在任何开发前收集项目背景，形成 AI 长期协作上下文
- [阶段门禁评审](./02_stage_gate_reviewer/SKILL.md)：每个阶段结束前检查产出是否足够进入下一阶段，防止返工
- [MVP 范围控制](./03_scope_controller/SKILL.md)：帮助单人开发者砍需求，防止项目过大、延期、失控
- [需求变更管控](./04_change_request_manager/SKILL.md)：把临时改需求变成正式变更单，评估影响并同步文档
- [文档交付管理](./05_document_deliverable_manager/SKILL.md)：管理 PRD、接口、数据库、部署、用户手册等交付文档
