# 02_product_design Skills

本目录包含以下细粒度 Skill：

- [信息架构设计](./00_information_architecture/SKILL.md)：设计菜单、页面层级、模块结构和入口关系
- [用户流程设计](./01_user_flow_design/SKILL.md)：设计核心用户路径和关键转化流程
- [低保真原型说明](./02_wireframe_spec/SKILL.md)：生成可交给 Figma/Axure 或 AI 设计工具的原型描述
- [交互状态设计](./03_interaction_state_spec/SKILL.md)：定义加载、空数据、错误、禁用、成功、失败状态
- [表单设计规则](./04_form_design_rules/SKILL.md)：定义字段、校验、错误提示、提交、防重复逻辑
- [列表筛选分页设计](./05_list_filter_pagination_design/SKILL.md)：设计列表字段、筛选条件、排序、分页、批量操作
- [UI 风格规范](./06_ui_style_guide/SKILL.md)：定义主色、字体、按钮、卡片、弹窗、表单、间距规范
- [组件清单](./07_component_inventory/SKILL.md)：把页面转成可复用组件，减少重复开发
- [提示文案设计](./08_microcopy_prompt_text/SKILL.md)：生成错误、成功、确认、空状态、权限不足等文案
- [设计评审](./09_design_review/SKILL.md)：检查原型/UI 是否有逻辑、适配、开发复杂度问题
