# 01_requirements Skills

本目录包含以下细粒度 Skill：

- [想法转需求](./00_idea_to_requirements/SKILL.md)：把一句项目想法拆成用户、场景、功能、规则、限制
- [用户角色与权限分析](./01_user_role_permission_analysis/SKILL.md)：识别角色、数据范围、操作权限，避免后期越权
- [业务流程梳理](./02_business_process_mapping/SKILL.md)：画清用户从进入到完成目标的完整业务链路
- [MVP 范围定义](./03_mvp_scope_definition/SKILL.md)：确定第一版只做什么、不做什么、为什么
- [PRD 生成](./04_prd_generator/SKILL.md)：生成可直接用于开发的需求文档
- [页面功能说明](./05_page_function_spec/SKILL.md)：逐页拆解页面模块、按钮、字段、状态和跳转
- [业务规则定义](./06_business_rule_spec/SKILL.md)：定义积分、订单、退款、权限、审核、状态流转等规则
- [异常场景挖掘](./07_exception_scenario_mining/SKILL.md)：系统挖掘空值、重复提交、失败、权限不足、断网等情况
- [验收标准生成](./08_acceptance_criteria_builder/SKILL.md)：把需求转成可验收、可测试、可交付的标准
- [需求评审](./09_requirements_review/SKILL.md)：从产品、前端、后端、测试视角找需求漏洞
- [需求变更单](./10_requirement_change_order/SKILL.md)：对新增/修改需求形成正式变更流程
