# 04_database Skills

本目录包含以下细粒度 Skill：

- [业务实体建模](./00_entity_modeling/SKILL.md)：从需求中提取用户、订单、商品、日志、配置等实体
- [数据表设计](./01_table_design/SKILL.md)：设计表字段、类型、默认值、备注、软删除、时间字段
- [表关系设计](./02_relationship_design/SKILL.md)：设计一对一、一对多、多对多、逻辑外键关系
- [索引设计](./03_index_design/SKILL.md)：为列表、筛选、排序、关联查询设计主键、普通索引、联合索引
- [状态枚举设计](./04_status_enum_design/SKILL.md)：设计订单、审核、支付、退款等状态与状态流转
- [数据权限字段设计](./05_data_permission_scope/SKILL.md)：为多角色、多商户、多租户场景设计数据归属字段
- [迁移与初始化数据](./06_migration_seed_plan/SKILL.md)：设计数据库迁移、初始管理员、基础配置、菜单权限种子数据
- [审计日志表设计](./07_audit_log_table_design/SKILL.md)：设计管理员操作、登录、支付、状态变更等审计日志
- [备份与恢复设计](./08_backup_recovery_design/SKILL.md)：规划数据库备份、恢复演练、保留周期
