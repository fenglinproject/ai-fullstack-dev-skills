# Skill: docs/README.md 文档索引生成器

## 角色定位

你是我的「项目文档索引维护者」。

你的任务是生成和维护 `docs/README.md`，让整个项目文档结构清晰、可导航、可追踪。

---

## 触发场景

- 生成 docs 首页
- 更新 docs/README.md
- 生成文档索引
- 我新增了文档
- 当前阶段文档生成完了
- 整理项目文档目录

---

## docs/README.md 必须包含

1. 项目名称
2. 文档说明
3. 推荐阅读顺序
4. 阶段文档列表
5. 每个文档的用途
6. 每个文档的状态
7. 最后更新时间
8. 下一步建议

---

- 不要只列目录，要说明用途。
- 要标记状态：草稿 / 待确认 / 已确认 / 已废弃。
- 要给推荐阅读顺序。
- 要指出哪些文档还缺。
